# ScriptVault
A full-stack script distribution platform built with Node.js and Express, featuring script uploads, category filtering, a rating system, and a client-side injection engine.
 #  Script Marketplace

Mini App Store for JavaScript scripts.

## Features
- Upload scripts
- Browse scripts
- Categories
- Ratings
- One-click inject

## Run

npm install
node server/server.js

Open http://localhost:3000
